## ====================================
The SecureOCR System is a Python-based solution for performing **high-accuracy Optical Character Recognition (OCR)** on sensitive documents. Crucially, the system is designed to run **completely offline** within a secure Linux environment (like **WSL**), ensuring **zero data leakage** risk to the cloud or external services. The system features advanced image pre-processing (using **OpenCV**) for maximum accuracy and includes a mandatory **secure file deletion** step to protect sensitive input data after processing.

## ====================================
## SCRIPT USAGE
## ====================================
1. Production Script: ocr_master_wsl.py
This script is for batch processing production documents. It handles all security and quality checks automatically.

Workflow
Place all sensitive images (.jpg, .png, .tiff, etc.) into the ./input_images/ folder. Run the script in your activated virtual environment.

```bash
python3 ocr_master_wsl.py
```

Output and Security
Results: Extracted text files (.txt) are saved to the ./output_text/ directory.

Security: The script performs a 3-pass secure overwrite and deletion of the original input files in ./input_images/. This folder will be empty after a successful run.

Training Script: interactive_trainer_wsl.py
This script is used to improve the system's accuracy on specific, difficult document types by creating clean training data.

Workflow
Place ONE difficult or representative image into the ./input_images/ folder. Run the training script.

```bash
python3 interactive_trainer_wsl.py
```
